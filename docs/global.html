multiColSort(orderby) → {function}

Source:
  • queryUtils.js, line 55

Helper function. It accumulates an array of asc or desc calls to implement multi column sorting for local stores. It is usually used internally and should not be used outside of building custom models.

Example
// multi column sorting
[
 {a: 1, b:1}, 
 {a:1, b: 2}, 
 {a: 2, b:1}, 
 {a: 2, b: 2}
].sort(multiColSort([asc('a'), desc('b')]))
// result:
[
 {a: 1, b: 2},
 {a: 1, b: 1},
 {a: 2, b: 2},
 {a: 2, b: 1}
]
Parameters:
Name Type Description
orderby Array

An array or single asc/desc function to accumulate
Returns:

A comparison function which combines the provided array of sort functions.

Type
function
